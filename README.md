# 🌸 Berry Bloom Cafe

A responsive front-end website for a strawberry-themed cafe. This project was created as a university project to practice responsive web design, custom styling, and page structuring using HTML and CSS.

---

## ✨ Features

- **Fully Responsive Design:** Optimized for different screen sizes and devices.
- **Interactive Image Slider:** Smooth and dynamic image slider powered by WowSlider.
- **Organized Sections:** Includes Home, About, Menu, Testimonials, Gallery, and a Book a Table reservation form.
- **Themed UI:** Elegant color palette and visual style inspired by the cafe's strawberry theme.

---

## 🛠️ Tech Stack & Tools

- **HTML5:** Semantic page structure and content.
- **CSS3:** Styling, layouts, and responsive design.
- **WowSlider:** Dynamic image slider component.
